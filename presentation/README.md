## Tech Stack (Everyone Should Know)

- **Frontend**: React.js, Material UI, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: SQL Server with Sequelize ORM
- **Auth**: JWT (JSON Web Tokens) + bcrypt
- **Email**: Nodemailer with HTML templates
- **File Upload**: Multer (resume + profile pictures)
- **Scheduling**: node-cron (daily deadline reminders)
